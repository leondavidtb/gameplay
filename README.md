## Pré-requisitos:

- Node.js
- Yarn
- Expo CLI

## Como utilizar:

### App

Clone o repositório

```bash
# Abra o terminal e digite esse comando ou faça download do projeto
$ git clone https://github.com/leondavidtb/gameplay.git
```

Entre na pasta

```bash
$ cd gameplay
```

Instale as dependências

```bash
$ yarn install
```

Inicie a aplicação

```bash
$ expo start
```
